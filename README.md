# 奇迹挂机 - 竖屏放置手游

基于策划案《竖屏挂机奇迹手游策划案（GUI驱动、极简战斗、不休结构）》开发的竖屏挂机游戏。

## 项目描述

- **类型**: 竖屏挂机、放置养成、装备合成
- **核心玩法**: 挂机刷装 + 奇迹式强化/合成/卓越/翅膀 + 不休式界面结构
- **目标**: 轻量、快速迭代、低美术成本、高留存

## 核心系统

### 1. 战斗系统
- 极简2D/纯文字表现
- GUI驱动，不做复杂动画
- 靠界面数值、飘字、光效反馈

### 2. 装备系统（奇迹风格）
- 9部位：武器、头盔、铠甲、护腿、护手、靴子、项链、戒指×2、翅膀
- 6品质：白、绿、蓝、紫（卓越）、橙、红
- 1~6阶随地图关卡掉落

### 3. 合成系统
- **强化** +1~+9：祝福宝石、灵魂宝石
- **突破** +10~+15：核心爽点
- **进化**：普通→卓越
- **翅膀**：奇迹标志性系统
- **宝石**：3合1
- **套装**：3件卓越激活

### 4. 挂机系统
- 全自动攻击、自动拾取、自动回血
- 离线收益最长24小时
- 快速战斗：看广告/消耗道具立即获取收益

## 界面结构

完全参考不休乌拉拉布局：
- 顶部栏：角色头像、等级、战力、钻石/金币
- 中部：战斗区（角色+怪物头像+血条+伤害飘字）
- 底部常驻TAB：首页/冒险/背包/合成/角色/福利
- 右侧悬浮：快速收益、一键强化

## 技术栈

- 纯HTML/CSS/JavaScript
- 单文件实现，便于部署
- 响应式竖屏适配

## 如何运行

直接用浏览器打开 `index.html` 即可游玩。

局域网访问：启动本地服务器后可通过局域网IP访问。
